About

Hello! I'm Josien Brands and I'm working on becoming the most awesome front-end web developer I can be! The learning never stops, so it only gets better, right? Coding has been trying to get my attention for some time now but it wasn't until last year I stoppped, listened and decided to give it a serious try. Finally!

The only regret so far is not listening before, coding web sites is great, I love it! This seems to be the thing where all my talents, skills and interests meet. So I spend as much time as I can now working on my front-end development skills.

Right now I'm focussing on basic websites. And with basic I do not mean boring, no way! You can do amazing things with CSS (and ok, maybe add some JS/jQuery). What I mean with basic is you do not need a big multipage hyper interactive web project for a clear and noticeable online presence.

The most important thing is to BE online, in a way that fits you/your company/brand, and is accessible for today's multitude of devices. And I can provide that for you, how awesome is that! Not being online or being accessible is robbing yourself of potential clients, so get in touch with me and we'll figure out how to fix this.

So here I am, putting myself out there, looking for projects to help you, by making your web dreams come true, and to challenge myself, by actually making those dreams come true. Sounds good right? So contact me (twitter, mail, smoke signals, come by for a drink) and lets make the web a little better together!
